## Naming Convention

```
YYYYMMDD-HHMMSS_topic_slug.ext
```

- **Timestamp**: When the screenshot was captured
- **Topic**: Category/source (google-doodle, github, docs, etc.)
- **Slug**: Brief kebab-case description
