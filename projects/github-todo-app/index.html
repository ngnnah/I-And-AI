GitHub TODO App

Your todos, stored in your own GitHub repo. Learn more

First time? Follow these steps:

  1. Create a private repository (e.g., my-todos)
  2. Create a fine-grained PAT:
    • Token name: todo-app
    • Repository access: Only select repositories → select your todo repo
    • Permissions → Repository permissions → Contents: Read and write
  3. Copy the token and paste it above
Why fine-grained PAT? It's scoped to only your todo repo. Even if leaked, it can't access your other repositories.
