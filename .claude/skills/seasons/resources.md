# Seasons Resources

Reference for the 72 micro-seasons with English translations, images, and curated links.

## Image Management

### Naming Convention

Images stored in `./images/` with format: `{number}-{romaji-slug}.jpg`

Example: `70-fuki-no-hana-saku.jpg`

### Download Script

```bash
# Download all images from Unsplash
export UNSPLASH_ACCESS_KEY="your-key"
uv run download_images.py

# Download specific kō
uv run download_images.py 70 71 72
```

See `download_images.py` for search terms and configuration.
