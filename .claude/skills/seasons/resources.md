# Seasons Resources

This file contains curated images and URLs for each of the 72 micro-seasons.

## Image Naming Convention

Images are stored in `./images/` with the format: `{number}-{romaji-slug}.jpg`

Example: `70-fuki-no-hana-saku.jpg`

## How to Add Images

1. Find a suitable image from Wikimedia Commons or take a screenshot
2. Save to `./images/` with the naming convention above
3. Recommended size: 640px width (for fast loading)
4. Prefer images showing the specific plant/animal/phenomenon
