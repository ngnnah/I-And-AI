GitHub TODO App

Your todos, stored in your own GitHub repo. Learn more

⚠ Security Notice
  • Use a fine-grained PAT (not classic token)
  • Scope it to only your todo repository
  • Grant only Contents: Read and write permission
  • If token leaks, attacker can only access your todos - not other repos
⚠ Don't check this on shared or public computers

First time? Follow these steps:

  1. Create a private repository
    Example: my-todos or test-private-file-based-todo-app
  2. Create a fine-grained Personal Access Token:
    • Token name: todo-app
    • Expiration: 90 days (or custom)
    • Repository access: "Only select repositories" → select your todo repo only
    • Permissions → Repository permissions → Contents: Read and write
    ⚠ CRITICAL: Select Contents, NOT "Actions". These look similar but "Actions" won't work!
  3. Copy the token (starts with github_pat_) and paste above
